# Security Policy for Notion Docs

This document describes the security policy for this open source project.

## Supported Versions

Security patches are provided only for the latest release version. The current latest release is shown below.

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/lgwk42/notion-docs)](https://github.com/lgwk42/notion-docs/releases)

## Reporting Vulnerabilities

Security vulnerabilities can be reported via GitHub Issues or the email address listed below. If reporting through an issue, please follow the bug report template. If reporting via email, please use the following format.

**Email:** luisgunkim@gmail.com

1. Subject
2. Summary
3. Affected code or library link where the vulnerability occurs
4. Scope of impact
5. Steps to reproduce + logs
6. Attachments

## Confidentiality

Reporters will be anonymized upon request; otherwise, real names will be used by default. Reporters must not disclose or leak the vulnerability to the public before a patch is released. The reporter assumes full responsibility for any damages caused by violating this obligation.
