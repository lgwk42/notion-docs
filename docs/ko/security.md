# Notion Docs 보안 정책

이 문서는 본 오픈소스 프로젝트의 보안 정책을 설명합니다.

## 지원 버전

보안 패치는 최신 릴리즈 버전에 대해서만 제공됩니다. 현재 최신 릴리즈는 아래에서 확인할 수 있습니다.

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/lgwk42/notion-docs)](https://github.com/lgwk42/notion-docs/releases)

## 취약점 신고

보안 취약점은 GitHub Issues 또는 아래 이메일 주소를 통해 신고할 수 있습니다. 이슈를 통해 신고할 경우 버그 리포트 템플릿을 따라주시고, 이메일로 신고할 경우 아래 양식을 사용해 주세요.

**이메일:** luisgunkim@gmail.com

1. 제목
2. 요약
3. 취약점이 발생하는 코드 또는 라이브러리 링크
4. 영향 범위
5. 재현 절차 + 로그
6. 첨부 자료

## 비밀 유지

신고자는 요청 시 익명 처리되며, 별도의 요청이 없을 경우 실명이 기본으로 사용됩니다. 신고자는 패치가 배포되기 전에 해당 취약점을 공개하거나 유출해서는 안 됩니다. 이 의무를 위반하여 발생하는 모든 손해에 대한 책임은 신고자 본인에게 있습니다.
